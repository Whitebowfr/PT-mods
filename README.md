# Collection of mods for CoffeeHeaven's Processor Tycoon game
Hi ! I'm Whitebow, currently studying microelectronics. I found this game by pure chance, but now I'm addicted. I have acquired a bit of knowledge modding another game called PC Building Simulator a few years ago, and I thought I could apply it here.

If you have any fun ideas for mods, send me a PM on discord ! I don't have a lot of free time so I won't be making very extensive changes, but I can always add some tweaks.

# How to install
You will need to install BepInEx into the game folder. Once that's done, download the mod .dll and place it in BepInEx/plugins. (If it's not there, launch the game once and close it)

# QOLImprovements
For now, this mod only adds 3 things :
- In the products spreadsheet, when clicking on a CPU, it will open the Inspector with this CPU selected.
- Same on the Top Sales graph legend
- When opening the Inspector using these methods, a Compare button will appear, that will open a second instance of the Inspector.
